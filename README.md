# 2016 Nov. - Japan Tokyo 7 days schedules

### 注意事項
##### 手提行李
+ 可攜帶一個大件和一個小件手提行李，總重量不可超過 7 公斤
(包含免稅商品、紀念品等通過安檢後購入的物品)
+ 高寬厚限制: 56/36/23(硬外殼)  or 114/60/11(軟外殼)
+ 電池、行動電源、充電器必須放在手提行李
+ 可帶上飛機的小物品: 手袋、袖珍書或錢包、大衣、雨傘，或經批准的免稅商品（僅限國際航班）
+ 不可帶上飛機: 液體噴霧類/刀類/棍棒類()

##### 托運行李
+ 長寬高總和不得超過 203cm,每件限重 20kg
